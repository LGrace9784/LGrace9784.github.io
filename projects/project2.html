URL Shortener Service

Description

A robust backend service built with Spring Boot that provides URL shortening functionality with comprehensive click analytics. The service generates short codes for long URLs and tracks detailed usage statistics including click counts, IP addresses, user agents, and referrers.

Technologies Used

  • Java 17
  • Spring Boot 3.2.0
  • Spring Data JPA
  • H2 Database
  • Maven

Features

  • URL shortening with unique short codes
  • Automatic redirection to original URLs
  • Click tracking and analytics
  • RESTful API design
  • H2 in-memory database
  • CORS support for frontend integration
  • Input validation and error handling
View Documentation GitHub Repo

API Documentation & Demo

1. Shorten a URL

POST /api/shorten

Convert a long URL into a short URL.

Request Body: { "url": "https://www.example.com/very/long/url/that/needs/shortening" } Response: { "shortUrl": "http://localhost:8080/AbCdEf", "originalUrl": "https://www.example.com/very/long/url/that/needs/shortening", "shortCode": "AbCdEf" }

2. Access Shortened URL

GET /{shortCode}

Redirect to the original URL and record the click.

Example: GET /AbCdEf → Redirects to original URL

3. Get Analytics

GET /api/analytics/{shortCode}

Retrieve detailed click analytics for a shortened URL.

Response: { "shortCode": "AbCdEf", "originalUrl": "https://www.example.com/very/long/url/that/needs/shortening", "totalClicks": 42, "createdAt": "2024-01-01T12:00:00", "clicks": [ { "id": 1, "ipAddress": "192.168.1.1", "userAgent": "Mozilla/5.0...", "referrer": "https://google.com", "clickedAt": "2024-01-01T12:30:00" } ] }

Getting Started

To run this service locally:

1. Navigate to the backend directory: cd projects/url-shortener-backend 2. Run with Maven: mvn spring-boot:run 3. The service will be available at http://localhost:8080 4. Access H2 console at: http://localhost:8080/h2-console
